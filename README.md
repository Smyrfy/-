# Пояснення до завдання 3

У цьому завданні програма спочатку запитує у користувача кількість чисел, які потрібно ввести. Після цього користувач вводить ці числа одне за одним. Програма додає кожне введене число до загальної суми.

Для перевірки правильності вводу використовується перевірка: якщо користувач вводить не число, програма повідомляє про помилку і завершує роботу. В кінці виводиться сума всіх введених чисел.

# Пояснення до завдання 4

У цьому завданні створено функцію, яка обробляє рядок, що складається з великих літер і цифр.

Кожна літера перетворюється у число за алфавітом (A = 1, B = 2, …, Z = 26). Якщо кілька літер йдуть підряд, їх значення додаються між собою. Цифри залишаються без змін і просто додаються до результату.

Рядок обробляється по символах зліва направо. Якщо зустрічається некоректний символ (наприклад, знак або маленька літера), функція повертає повідомлення про помилку.

В кінці формується новий рядок із результатів перетворення і повертається як відповідь.
